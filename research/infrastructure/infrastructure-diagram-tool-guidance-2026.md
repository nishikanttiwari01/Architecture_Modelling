## Relevant summary

The listed project and vendor sources describe common tools and official icon libraries used to create infrastructure and cloud architecture diagrams. Chapter 11 uses them to help beginners distinguish drawing tools, model-based tools and diagrams-as-code approaches without prescribing one tool for every organisation.

## Terminology and version notes
