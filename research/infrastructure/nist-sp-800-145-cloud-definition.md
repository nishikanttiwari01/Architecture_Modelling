## Relevant summary

NIST SP 800-145 provides a concise definition of cloud computing and names essential characteristics, service models and deployment models. Chapter 11 uses it to keep cloud wording grounded in an official source while avoiding vendor-specific claims.

## Terminology and version notes

Use NIST cloud terminology when discussing public, private, community and hybrid cloud models. The source is older but remains a commonly cited baseline definition rather than current vendor implementation guidance.
