- Intended chapter(s): 51
- Normative or interpretive: Normative for OpenAPI scope; interpretive for contract-governance guidance.

## Relevant summary

OpenAPI describes HTTP API surfaces, operations, parameters, payloads, responses and related contract information. It supports discovery, documentation and tooling. Horizon Bank uses it as one possible contract representation for governed APIs, not as proof that an API is semantically correct, secure or operable.

## Terminology and version notes
