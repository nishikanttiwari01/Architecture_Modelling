## Relevant summary

The PCI Security Standards Council describes PCI DSS as a baseline for payment-account-data security. The scope description supports treating card-data boundaries and third-party responsibilities as explicit architecture concerns. It does not justify assuming that every card application contains the same data or that a catalogue entry proves compliance.
