# Reusable Codex Prompts

## Start a chapter

Read `AGENTS.md` and all mandatory control files. Prepare Chapter NN for drafting. Verify its dependencies, expand its outline, identify required primary sources and diagrams, create research tasks, and update `STATUS.md`. Do not draft unsupported factual claims.

## Draft a chapter

Read the project control files, the target chapter, neighbouring chapters and relevant research notes. Draft the requested sections in the repository style. Add source keys, update registers, run checks, and report all changed files. Do not mark the chapter Approved.

## Review a chapter

Review Chapter NN in separate passes for technical accuracy, beginner comprehension, educational sequence, terminology consistency, diagram quality and source quality. Save findings under the appropriate `reviews/` folder, update status, and correct only issues within the requested scope.

## Cross-book consistency check

Search the full manuscript for inconsistent definitions, duplicate explanations, conflicting examples, naming drift and unsupported version claims. Produce a prioritised review file and update only safe mechanical inconsistencies unless broader revision is explicitly requested.

## Build a diagram

Read the chapter and `DIAGRAM_REGISTER.md`. Confirm the figure purpose, audience, notation and abstraction level. Create an editable source under the correct diagram folder, export SVG, add accessible explanatory text and update the register.
