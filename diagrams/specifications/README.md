# Diagram Specifications

Create one specification per figure before creating diagram source.

Use `templates/diagram-specification-template.md` and name files with the figure ID, for example `FIG-05-01-c4-levels-of-zoom.md`.

A specification records the diagram purpose, audience, notation, required elements, relationships, scope, exclusions, source references and review criteria. Diagram source should not be created until the author has approved the specification.
