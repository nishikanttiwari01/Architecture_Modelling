# Diagram Style Guide

Use this folder for shared diagram styling rules and reusable rendering assets.

General rules:

- Prefer SVG for publication output.
- Use colour sparingly and never as the only carrier of meaning.
- Keep labels readable at intended book-page width.
- Avoid clipped text, overlapping labels and excessive line crossings.
- Keep terminology consistent with `GLOSSARY.md`.
- Make direction, ownership and boundaries explicit where relevant.

PlantUML diagrams should include `plantuml-theme.puml` unless a specific diagram needs a justified exception.
